# 🔆 Touch Sensor Controlled LED using Arduino

## 📌 Overview
This project demonstrates a touch-based control system using a capacitive touch sensor and Arduino.

## 🎯 Objective
- Detect human touch
- Control LED using Arduino
- Replace mechanical switches

## ⚙️ Components Used
- Arduino UNO
- TTP223 Touch Sensor
- LED
- 220Ω Resistor
- Breadboard & Wires

## 🔄 Working
- Touch sensor detects input
- Sends HIGH signal to Arduino
- Arduino turns LED ON/OFF

## 🧠 Algorithm
1. Initialize system
2. Read touch input
3. If touched → LED ON
4. Else → LED OFF
5. Repeat continuously
